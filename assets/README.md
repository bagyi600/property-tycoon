### 3. Asset Specifications:

#### Textures:
- **Format**: PNG (transparency) or JPEG
- **Size**: Power of 2 (128, 256, 512, 1024)
- **Color**: RGB/RGBA
- **Resolution**: 72 PPI for screen

#### Sounds:
- **Format**: OGG (preferred) or WAV
- **Sample Rate**: 44100 Hz
- **Channels**: Mono for SFX, Stereo for music
- **Length**: SFX: 0.1-3s, Music: 1-3 minutes

### 4. Installation:

1. Download assets
2. Replace placeholder files
3. Keep same filenames or update references in:
   - `project.godot`
   - GDScript files
   - Scene files

### 5. License Check:

- No redistribution restrictions
- Compatible with Godot engine
